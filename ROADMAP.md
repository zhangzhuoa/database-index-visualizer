# ROADMAP

## v1.0 已完成

当前项目已经整理为可部署的在线展示版：

- 纯前端静态网站，不新增后端、数据库或用户系统。
- 支持 B+ 树、哈希索引、R 树、四叉树和网格索引。
- 提供统一操作面板、内置示例、动画播放控制和事件说明面板。
- 已通过 Vercel 部署，可通过公网 URL 访问。
- 已补充 README、架构说明、AI Coding 说明和版本记录。

## 面试展示重点

- 多索引统一 `IndexEngine` 架构。
- 核心算法与 React 页面组件解耦。
- ViewModel 驱动 SVG 可视化。
- `AnimationEvent` 驱动逐步动画和事件说明。
- 使用 AI Coding 辅助需求拆解、测试补充、文档整理和上线准备。

## 后续可选扩展

这些不是 v1.0 必需内容，只作为后续可选方向：

- 保存和分享自定义演示案例。
- 支持 CSV/JSON 数据导入。
- 支持导出截图或录制动画。
- 增加更多布局策略和视觉主题。
- 增加后端服务保存案例，但这会改变当前纯前端定位。

## 不计划在 v1.0 中加入

- 真实数据库连接。
- 用户登录系统。
- 后端服务。
- D3、React Flow、Cytoscape 等复杂图形库。
- 新的索引结构。
